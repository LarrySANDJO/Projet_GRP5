Bien vouloir lire ceci avant de consulter les éléments du projet.

1) Structure du Projet

Le projet est constitué de sept dossiers :

- application beamer et html
- Beamer
- Document_Rmarkdown
- Images
- Livre
- Publipostage
- Sorties

2) Détails sur les dossiers

- Le dossier 'application beamer et html' contient les applications beamer et html qui présentent respectivement la creation de beamer ainsi que les possibilités qu'offrent les beamers et les possibilités de sorties avec html. Pour le beamer, veuillez regarder les fichiers 'work_COD' ensuite 'beamer_introduction', et enfin 'beamer_code'. En outre, le fichier 'html' vous présente comment personnaliser son document html.


- Le dossier Beamer contient la présentation beamer qui présente l'essentiel des informations à savoir en ce qui concerne les documents Rmarkdown ainsi que le fichier Rmd de ce beamer qu'il est possible de compiler.

- Le dossier Document_Rmardown contient le document pdf et le document Rmd à compiler. Le document Rmardown nommé "Document_RMD_GRP5" contient tous les détails nécessaires pour maitriser la création de documents Rmarkdown. Il contient aussi toutes les mises en formes décrites dans le document (Entête, pieds de pages, pages de garde,...)

- Le dossier 'Images' contient les différentes images qui ont été utilisés pour le travail.

- Le dossier Livre contient le projet de création du livre à l'aide de bookdown. Le livre se trouve dans '_book' et est nommé '_main'. Nous avons renommé ce livre dans le dossier des sorties comme étant 'Livre(Exemple)'.

- Le dossier Publipostage contient le modèle word de publipostage utilisé pour montrer l'utilité de l'option 'param' de R markdown. Ce dossier contient le script à compiler pour voir les différentes sorties qui se trouvent dans le dossier 'Sortes_publipostage'.

- Le dossier Sorties contient toutes les sorties présentent dans tous les dossiers si vous souhaitez visualiser.

3) Documents les plus importants

Les documents les plus importants à consulter dans ce projet son dans le dossier 'Sorties'.
Pour les consulter, nous vous proposons l'ordre suivant :

- Commencer par le document Document_RMD_GRP5 qui contient la majorité des détails.
- Visiter ensuite le document beamer_introduction pour savoir comment fonctionnent les beamers.
- Ouvrez ensuite beamer_code pour voir comment on créer un beamer ainsi que les types de sorties possibles.
- Vous pourrez alors consulter le beamer Beamer_GRP5 qui résume les éléments du document global Document_RMD_GRP5.
- Après cela, vous ouvrez le fichier 'html.html' qui vous explique comment personnaliser son document Rmarkdown pour les sorties html.(très intéressant)
- Continuez en ouvrant le fichier application_html pour voir les sorties possibles avec les documents html
- Vous pourrez ensuite consulter l'exemple de livre fait avec bookdown.
- Enfin dans le dossier 'Sorties_publipostage', vous verrez les oeuvres de l'utilisation de l'option param de Rmarkdown.
